# F401CCU6_Dev

此项目的目标是做一个引出所有IO，以实现一个F401CCU6开发板的效果。参考了网上常见的板型，加入了RGB灯。

该项目MCU采用的STM32F1401CCU6，偶然翻出几年前留下的几枚mPOS也是画这板子的原因之一。该板设计了RST、BOOT按键，并引出了SWD调试接口，同时为了迎合时代发展，USB采用了Type C接口。
